# 🅳 串流計算 / 儲存分發

## 共通模式
高流量讀寫，需要**大規模儲存**、**聚合計算**與**全球分發**（CDN / 快取）。

## 可複用心法
- **無狀態服務水平擴展** + 負載均衡；ID 生成（雪花 / 短碼 / 雜湊）。
- **快取分層**：CDN → 反向代理 → 應用快取（Redis）→ DB；快取失效策略。
- **物件儲存**：大檔（圖片/影片）放 S3 類儲存，DB 只存中繼資料。
- **串流聚合**：Top-K / 趨勢用 **Count-Min Sketch**、**滑動視窗**、Kafka + Flink；近似演算法換取吞吐。
- **讀寫分離 / 分片**：依 key 分片，讀副本擴展讀流量。
